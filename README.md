# Praktikum Basis Data I (Database Practices I)

Repositori ini adalah koleksi lengkap dari hasil praktik dan dokumen tugas yang dibuat untuk mata kuliah **Praktikum Basis Data I**. Repositori ini berfokus pada pemahaman dan implementasi konsep dasar Basis Data Relasional, mulai dari perancangan hingga manipulasi data menggunakan SQL.

### üìö Tujuan Proyek

Tujuan utama dari repositori ini adalah untuk:
1.  Mendokumentasikan semua laporan dan hasil ujian (Tes Awal/Akhir) dari setiap bab praktikum.
2.  Menyimpan skrip SQL yang digunakan untuk membuat skema basis data dan menguji kueri.
3.  Menunjukkan pemahaman tentang Structured Query Language (SQL) dan konsep Basis Data.

---

### üíª Teknologi dan Alat

Proyek-proyek dan skrip dalam repositori ini menggunakan teknologi berikut:

| Kategori | Detail |
| :--- | :--- |
| **Sistem Basis Data (RDBMS)** | MySQL / Oracle (Asumsi, umum digunakan dalam praktikum) |
| **Bahasa Kueri** | SQL (Structured Query Language) |
| **Dokumentasi** | Microsoft Word (`.docx`), PDF (`.pdf`) |

---

### üìù Topik dan Bab yang Dicakup

Repositori ini mencakup berbagai topik Basis Data yang terbagi dalam beberapa bab, yang sebagian besar diwakili oleh laporan Tes Awal dan Tes Akhir.

| Bab | Topik Utama yang Dicakup (Inferred) | Skrip SQL Terkait |
| :--- | :--- | :--- |

| **Bab 4 & 5** | Data Manipulation Language (DML): `INSERT`, `SELECT`, `UPDATE`, `DELETE`. Fungsi Dasar. | `notepad.sql` |
| **Bab 6** | Filtering Data (`WHERE`), Ordering Data (`ORDER BY`), Fungsi Agregat. | |
| **Bab 7** | Menggabungkan Data (JOIN): Inner Join, Outer Join. Set Operations. | |
| **Bab 8** | Subkueri (Subqueries), Grouping Data (`GROUP BY`, `HAVING`). | |
| **Bab 9** | Konsep Lanjutan: Views, Indexes, Keamanan, atau Transaksi. | |
